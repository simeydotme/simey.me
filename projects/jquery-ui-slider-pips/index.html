Jquery UI Slider Pips

An extension for the jQuery UI slider widget

Tweet

Basic Example

Scroll down for more examples


  • Adds small "pips" and "labels" to the slider
  • Extends the jQuery UI Slider widget
  • Easily Configurable
  • Works for both Horizontal and Vertical sliders
  • Doesn't change default slider behaviour
  • Purely a visual plugin
  • IE7+, modern browsers.

Installation

First of all download the /src/ code from: GitHub


jQuery

Make sure you've included a jQuery & UI package with $.slider and module, I've shown them being loaded remotely, you can download and include the libraries yourself if you wish.
jQuery | UI

<script src="http://code.jquery.com/jquery-1.10.2.min.js"></script>
<script src="http://code.jquery.com/ui/1.10.3/jquery-ui.min.js"></script>

Include

Next we need to include the Pips Plugin. (dev or minified)

<script src="css/jquery-ui-slider-pips.min.js"></script>
<!-- <script src="js/jquery-ui-slider-pips.js"></script> -->

Stylesheets

Make sure you've got the stylesheets loading for the pip plugin.

<link rel="stylesheet" type="text/css" href="css/jquery-ui-slider-pips.css">

Initialize

All that's left now is to initialize the plugin.

At it's most basic you can just run it on a already-initialized slider:

$(document).ready(function() {
  $('div.slider').slider().slider('pips');
});

Options

Below are the plugins's default options which can be overidden by supplying an object to the plugin call.

// The Slider's Default Options:
$('div.slider').slider('pips', {

  first: "label", 	
  // string, boolean
  // eg: "label", "pip" , false
  // determines label for first item on slider.
  
  last: "label",
  // string, boolean
  // eg: "label", "pip" , false
  // determines label for last item on slider.

  rest: "pip", 		
  // string, boolean
  // eg: "label", "pip" , false
  // determines labels for all other items on slider.
  
  labels: false,
  // array
  // eg: ["Mon","Tues","Weds","Thurs","Fri","Sat","Sun"]
  // provides overrides for the labels so the
  // implementer can visually display a different
  // set of labels to the normal value
  
  prefix: "",			
  // string
  // eg: "$"
  // allows implementer to show a prefix value before
  // the values on the slider
  
  suffix: ""			
  // string
  // eg: ".00¢"
  // allows implementer to show a suffix value after
  // the values on the slider

});

The plugin also has a method for "floats"

// The Slider's Default Options:
$('div.slider').slider('float', {

  handle: true,
  // boolean
  // eg: true, false
  // activates the floats on slider handles

  labels: true,
  // boolean
  // eg: true, false
  // activates the floats on slider labels

  prefix: "",
  // string
  // eg: "$"
  // allows implementer to show a prefix value before
  // the values on the float

  suffix: ""
  // string
  // eg: ".00¢"
  // allows implementer to show a suffix value after
  // the values on the float

});

Styling

The default styles are pretty robust, but you can override them yourself by adding your own styles in your own stylesheet, or editing the one supplied.


Examples

Here is a list of examples with the code used.

$('.slider1').slider('pips');

The default configuration of the points.
NB: All sliders have been initialised by jQuery UI first


$('.slider2').slider('pips', { 
	rest: 'label' 
});

Here we tell the rest of the points to be labels.


$('.slider3').slider('pips', { 
	rest: false 
});

Here we ask the plugin to hide the rest of the points. Example also shows working with range.


$('.slider4').slider('pips', { 
	first: 'pip', 
	last: 'pip'
});

Here we ask the plugin to show pips for the first and last point (no labels).


$('.slider41').slider('pips', { 
	rest: 'label',  
	prefix: '$' , 
	suffix: '.00'
});

Here we have all labels, but crucially we supply a prefix and suffix.
NB: both prefix and suffix are optional and may require slight css tweaks


$('.slider21').slider('pips', { 
	rest: 'label', 
	labels: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'] 
});

Here we tell the rest of the points to be labels.
We are also assigning custom labels for each pip.


$('.slider5').slider('pips', { 
	rest: false 
});

Here are hiding the rest of the points again. Example of vertical slider.


$('.slider6').slider('pips', { 
	first: 'pip', 
	last: 'pip' 
});

Here we ask the plugin to show pips for the first and last point, the rest are defaulted.


$('.slider7').slider('float');

We can also have a slider with floating labels


$('.slider8').slider('pips').slider('float');

If we've already got pips, we can float those, too. (hover the pips).


$('.slider9')
	.slider('pips', { 
		rest: false, 
		prefix: "°", 
		suffix: "c" 
	}).slider('float', { 
		labels: false, 
		prefix: "°" , 
		suffix: "c" 
	});

And here's an example of the prefix and suffix on both the pips and the floats.

NB: both prefix and suffix are optional and may require slight css tweaks
