ui Slider Pips

/*
* a small plugin to extend the jQuery UI Slider widget, it
* adds little "pips" or "markers" on the slider, and 
* also it can add "floating" labels.

* plugin is purely visual, but many cool things can be achieved
* with it, check out the demos and docs.
*/

How to get it

// download it from github 
   https://github.com/simeydotme/jQuery-ui-Slider-Pips/tree/master/dist

// clone the repository
   $ git clone https://github.com/simeydotme/jQuery-ui-Slider-Pips.git

// install via bower
   $ bower install jquery-ui-slider-pips

Unorphanize

/*
* unorphanize allows us to remove widows/orphans (i like to say "kill")
* from any DOM elements by running this simple jquery method.
*
* it might not be as fast as other methods, but that's because
* it is way more robust, no worrying about edge-case problems! :)

* (I'm using on on headings for this website!)
